# Hand Gesture Recognition using MediaPipe
SkillCraft Technology — Machine Learning Internship Task 4

## What it does
Real-time hand gesture recognition using webcam that detects 6 gestures:
- ✊ Fist
- 🖐️ Palm  
- 👍 Thumbs Up
- ✌️ Peace
- 🤟 ILoveYou
- ☝️ Pointing

## Tech Stack
Python, MediaPipe, OpenCV, scikit-learn, matplotlib

## How to run
pip install mediapipe opencv-python scikit-learn matplotlib
python hand_gesture.py
